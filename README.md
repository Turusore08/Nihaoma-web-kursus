# Nihaoma - Kursus Bahasa Mandarin

Sebuah website sederhana untuk kursus bahasa Mandarin dengan desain yang menarik dan interaktif.

## Fitur
- **Desain Responsif**: Tampilan optimal di desktop maupun mobile.
- **Navigasi Mudah**: Akses cepat ke setiap bagian halaman.
- **Detail Kursus Interaktif**: Klik pada setiap level untuk melihat kurikulum dan harga.
- **Animasi Transisi**: Efek `fadeIn` yang halus saat membuka detail kursus.
- **Tema Warna Menarik**: Kombinasi warna merah Mandarin, emas, dan abu-abu.

## Struktur Proyek
- `index.html`: File utama berisi struktur halaman dan konten.
- `style.css`: File CSS untuk styling dan layout.

## Cara Menjalankan
1. Buka file `index.html` menggunakan browser web modern (Chrome, Firefox, Edge, Safari).
2. Klik pada setiap item kursus untuk melihat detailnya.
3. Jelajahi halaman menggunakan navigasi di header.

## Teknologi yang Digunakan
- HTML5
- CSS3
